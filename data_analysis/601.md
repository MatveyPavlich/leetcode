# SQL

## My solution:
```
WITH numbered_rows AS (
    SELECT
        *,
        id - ROW_NUMBER() OVER(ORDER BY visit_date) AS ranking
    FROM stadium 
    WHERE people >= 100
),

ranking AS (
    SELECT
        *,
        COUNT(id) OVER(PARTITION BY ranking) AS n_rows

    FROM numbered_rows
)

SELECT id, visit_date, people
FROM ranking
WHERE n_rows >= 3
ORDER BY id ASC
```
